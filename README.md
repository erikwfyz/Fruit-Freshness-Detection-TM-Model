##
🍎 Fruit Freshness Detection AI Model

An image classification system developed using Google Teachable Machine that automatically identifies whether a fruit is fresh or rotten based on visual characteristics.

This project demonstrates the application of Machine Learning and Computer Vision techniques to solve a real-world food quality assessment problem. The model was trained on a dataset containing 500+ fruit images representing various freshness conditions.

📌 Project Overview

The goal of this project is to assist in fruit quality inspection by providing a fast and automated method for detecting fruit freshness through image recognition.

The model learns visual patterns such as:

Color changes
Surface texture variations
Signs of decay
Physical appearance differences between fresh and rotten fruits
🚀 Features
Fresh vs. Rotten fruit classification
Image-based prediction
User-friendly deployment through Google Teachable Machine
Real-time webcam/image recognition support
No coding required for model testing
🛠 Technologies Used
Google Teachable Machine
TensorFlow.js
Machine Learning
Computer Vision
Image Classification
📊 Dataset
Over 500+ labeled fruit images
Categories:
Fresh Fruit
Rotten Fruit

The dataset was collected and organized to provide diverse examples for training and evaluation.

🎯 Model Performance

The trained model achieved high classification accuracy during testing within the Teachable Machine environment.

Note: Performance may vary depending on image quality, lighting conditions, camera angle, and fruit appearance.
